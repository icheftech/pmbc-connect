# PMBC Connect Documentation

## Overview
PMBC Connect is a comprehensive church management platform for Pentecostal Missionary Baptist Church.

## Project Structure
- `/frontend` - Next.js web application
- `/backend` - Node.js/Express API server
- `/docs` - Project documentation
- `/.github/workflows` - CI/CD pipelines

## Getting Started
See the README.md in the root directory for installation and setup instructions.

## Features
- Sermon streaming and archive
- Bible study materials
- Event management
- Pastoral care scheduling
- Community engagement tools
- Online giving integration
